# mapChar

## Description

The mapChar() function iterates over each character of the string and executes the provided callback function for each one.

## Install 

```
npm i map-char
```

## Import

```Javascript
require('map-char');
```

## Usage 

```Javascript
const str = 'test'

str.mapChar((char, i) => {
    console.log(char, i);
})
```
